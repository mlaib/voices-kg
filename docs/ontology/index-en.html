VOICES Ontology: Description back to ToC

Unified VOICES ontology for Holocaust survivor testimony knowledge graphs. Thesaurus-free public variant. External standards: CIDOC-CRM, OWL-Time, MFOEM, Web Annotation, PROV-O, Media Ontology.
